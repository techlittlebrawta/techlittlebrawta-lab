## Legacy securitylab archive

The remaining content from the former `techlittlebrawta/securitylab` repository
is preserved under [`legacy/securitylab/`](legacy/securitylab/). Original Nmap,
PNETLab, and Splunk files are byte-for-byte copies; separate documentation
explains requirements, operation, risks, validation, and recovery.

The obsolete Ansible Automation Platform 2.4 material was intentionally not
retained because the current [`ansible/`](ansible/) implementation supersedes
it.
