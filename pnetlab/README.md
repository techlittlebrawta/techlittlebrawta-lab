# PNETLab

This section contains installation and API automation for PNETLab lab systems.

## Contents

- [`pnetlab-v6-local/`](pnetlab-v6-local/) — installs PNETLab 6.0.0-103 and
  iShare2 on a fresh Ubuntu 20.04.6 AMD64 system.
- [`api-tools/`](api-tools/) — authenticated command-line tools for listing,
  adding, starting, stopping, and wiping nodes in an active PNETLab 6 session.

Read the README inside the selected project before running it. PNETLab node
operations can consume significant resources or remove saved node state, so test
automation in a disposable lab first.
