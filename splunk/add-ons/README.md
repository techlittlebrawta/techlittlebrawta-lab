# Splunk technology add-ons

Technology add-ons in this directory collect or normalize external data for
Splunk. Each add-on is self-contained and has its own installation, credential,
validation, and troubleshooting instructions.

## Available add-ons

- [`TA-veracode/`](TA-veracode/) — collects Veracode application findings as JSON events.

Review all scripts and configuration defaults before deploying an add-on. Keep
site-specific settings and credentials out of Git.
