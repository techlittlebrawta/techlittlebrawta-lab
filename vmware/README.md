# VMware lab automation

This directory contains VMware automation retained specifically for a private,
disposable training lab.

## Lab-only scope

- Use this material only on ESXi hosts you own or are authorized to administer.
- Do not use it on production, customer, employer, hosted, or shared systems.
- Take a recoverable host backup or hypervisor snapshot before running it.
- Comply with the VMware/Broadcom license terms that apply to the environment.
- Use the vendor-supported licensing interface and a valid license for systems
  that are not disposable lab hosts.
